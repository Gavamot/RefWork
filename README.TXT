файл содержит URL
один URL на строке
обработка подразумевает : 
- отправка запроса по URL 
- получение ответ
- подсчёт кол-ва ссылок на странице указанного URL (теги <a>)
отобразить результат в виде списка URL и число (кол-во ссылок) 
выделить размером шрифта строки с максимальными результатами 
во время обработки файла отображать некий progress bar 
необходимо использовать паттерн MVVM, Task'и и соответствующие
